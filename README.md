# Restaurant Website Template

A modern, responsive restaurant website built with Next.js and TailwindCSS.
This template includes essential features for restaurants such as table reservations, menu showcase, contact information, and gallery sections.

The goal of this project is to provide a professional website structure that restaurants can use to present their brand online and allow customers to easily reserve tables or contact the business.

---

## Features

* Modern responsive UI
* Table reservation form
* Menu showcase section
* Restaurant gallery
* Contact section with phone, email, and location
* WhatsApp booking integration
* Google Maps location support
* Mobile-friendly design
* Clean and modular component structure

---

## Tech Stack

* Next.js
* React
* Tailwind CSS
* TypeScript
* Shadcn UI components

---

## Project Structure

app/

* page.tsx – homepage
* menu/ – menu page
* gallery/ – gallery page
* contact/ – contact page

components/

* sections/ – main website sections
* navigation.tsx – navigation bar
* footer.tsx – website footer

## Getting Started

Clone the repository:

```bash
git clone https://github.com/0xpandey/Restaurant-template.git
cd Restaurant-template
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```

## Deployment

The project can be deployed easily using platforms like Vercel.

Steps:

1. Push the project to GitHub.
2. Import the repository into Vercel.
3. Deploy the project.

After deployment you will receive a public link that can be shared with clients or users.

---

## Customization

This template can be easily adapted for different restaurants by editing:

* restaurant name
* menu items
* contact information
* images
* reservation details

Most content can be modified in the component files inside the `components/sections` directory.
